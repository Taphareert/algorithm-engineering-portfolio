# Java Algorithms Collection

ชุดแบบฝึกหัดและตัวอย่างอัลกอริทึมที่พัฒนาด้วยภาษา Java  
มุ่งเน้นการฝึกตรรกะพื้นฐาน โครงสร้างข้อมูล และแนวคิดเชิงอัลกอริทึม

---

##  จุดประสงค์

- ฝึกการคิดเชิงอัลกอริทึม (Algorithmic Thinking)
- เข้าใจ Recursion และ Iteration
- ฝึกการทำงานกับ Linked List / Stack
- ฝึก Expression Parsing (Infix → Postfix)
- เข้าใจพื้นฐานโครงสร้างข้อมูลและ Complexity

---

##  โครงสร้างโปรเจค

###  GCD
- การหา Greatest Common Divisor
- ฝึกการใช้ Euclidean Algorithm

###  IF (Infix to Postfix & Expression Evaluation)
- แปลง Infix เป็น Postfix
- คำนวณค่า Postfix Expression
- ฝึกการใช้ Stack และ Linked List

###  Prime0
- การตรวจสอบจำนวนเฉพาะ
- โครงสร้าง Node / Linked List

###  TowersOfHanoi
- ตัวอย่าง Recursion แบบคลาสสิก
- แสดงลำดับขั้นตอนการย้ายแผ่น

---

##  สิ่งที่ได้เรียนรู้

- Recursion vs Iteration
- Stack-based expression evaluation
- Linked List implementation
- Algorithm complexity พื้นฐาน
- การแยก logic ออกจาก main method

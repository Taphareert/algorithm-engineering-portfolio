Prime Number Checker (Java)
    Project Overview

    โปรเจคนี้พัฒนาอัลกอริทึมสำหรับตรวจสอบว่า :
        จำนวนเต็มหนึ่งเป็น “จำนวนเฉพาะ (Prime Number)” หรือไม่
        โดยเน้นการออกแบบ Algorithm ให้มีประสิทธิภาพมากกว่าการตรวจสอบแบบพื้นฐาน
        พัฒนาโดยใช้ภาษา Java

    แนวคิดของอัลกอริทึม :
        แทนที่จะตรวจหารทุกจำนวนตั้งแต่ 2 ถึง n-1
        ระบบจะตรวจสอบเพียงถึง √n เท่านั้น

        เนื่องจาก:
            ถ้า n มีตัวหารที่มากกว่า √n
            จะต้องมีอีกตัวที่น้อยกว่า √n เสมอ
            ดังนั้นจึงลดจำนวนรอบการตรวจสอบได้อย่างมาก

    Algorithm Logic :
        ถ้า n ≤ 1 → ไม่ใช่จำนวนเฉพาะ
        วนลูปตั้งแต่ 2 ถึง √n
        ถ้าหารลงตัวแม้แต่ครั้งเดียว → ไม่ใช่จำนวนเฉพาะ
        ถ้าไม่พบตัวหารเลย → เป็นจำนวนเฉพาะ

    Complexity :
        Time Complexity: O(√n)
        Space Complexity: O(1)
        มีประสิทธิภาพมากกว่าการตรวจแบบ O(n)

    Technology :
        Java
        Control Flow
        Loop Optimization
        Mathematical Reasoning

    จุดประสงค์ของโปรเจค :
        ฝึกคิดเชิงตรรกะ 
        เข้าใจการปรับปรุงประสิทธิภาพของอัลกอริทึม
        เข้าใจแนวคิด Optimization พื้นฐาน
        สร้างพื้นฐานก่อนเรียน Algorithm ขั้นสูง

# แบบฝึกหัด Algorithm ด้วย Python

Repository นี้รวบรวมแบบฝึกหัดด้านตรรกะและอัลกอริทึมและแนวคิดเชิงโครงสร้างข้อมูล
ที่พัฒนาขึ้นระหว่างการศึกษาด้วยตนเอง

จุดประสงค์หลักคือฝึกพื้นฐานสำคัญของการเขียนโปรแกรม เช่น

- การแยกปัญหา (Problem Decomposition)
- การคิดเชิงอัลกอริทึม
- การใช้ Dictionary / HashMap
- การจัดการ String
- การตรวจสอบข้อมูล (Validation)
- การจัดโครงสร้างฟังก์ชันให้ชัดเจน

---

##  หมวดหมู่

###  hashing
โจทย์ที่ใช้แนวคิด HashMap / Dictionary
- Anagram Checker
- Two Sum (แบบ O(n²) และ O(n))
- Manual Frequency Counter

### string_manipulation
การแปลงหรือจัดรูปแบบข้อความ
- Caesar Cipher (Encrypt / Decrypt)


### mini_projects
โปรแกรมขนาดเล็กที่มีการตรวจสอบข้อมูลและจัดรูปแบบผลลัพธ์
- Character Status Generator (พร้อม Validation)
  
---

##  แนวคิดที่ใช้

- HashMap optimization
- Dictionary counting
- String normalization
- Guard Clauses
- Separation of Concerns
- Error Handling ด้วย try/except

---

##  ตัวอย่างความแตกต่าง

Two Sum มีทั้งแบบ

- O(n²) — Nested Loop
- O(n) — ใช้ HashMap

เพื่อแสดงให้เห็นการพัฒนา Algorithm ให้มีประสิทธิภาพขึ้น

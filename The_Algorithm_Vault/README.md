The Algorithm Vault:

    โครงงานส่วนตัวสำหรับฝึกและทดลองพัฒนาอัลกอริทึม พร้อมแสดงภาพการทำงานแบบ Interactive

    ฟีเจอร์หลัก:
        แสดงการทำงานของ A* และ Dijkstra
        ระบบสร้าง Maze อัตโนมัติ
        การจำลอง Stack และ Linked List
        เครื่องมือ Caesar Cipher และ Anagram Checker
        ส่วนวิเคราะห์ Time Complexity
        Backend สำหรับดาวน์โหลดไฟล์ (FastAPI)

    เทคโนโลยีที่ใช้:
        Frontend:
            HTML/CSS
            Vanilla JavaScript (ES6)
            Object-Oriented Programming
            Algorithm Implementation (A*, Dijkstra, Recursion, Heap)
        Backend:
            Python
            FastAPI

    ไม่มีการใช้ Framework สำเร็จรูป
    เน้นเขียน Logic และโครงสร้างเองทั้งหมด

    วัตถุประสงค์:
        พัฒนาเพื่อฝึกทักษะด้านโครงสร้างข้อมูล อัลกอริทึม และการออกแบบระบบ

Interactive Algorithm Learning Platform :

    ภาพรวมโปรเจค :
        The Algorithm Vault คือโปรเจคเว็บแอปพลิเคชันเชิงการศึกษา
        ที่พัฒนาเพื่ออธิบายและแสดงการทำงานของอัลกอริทึมและโครงสร้างข้อมูลแบบ Interactive

    เป้าหมายของโปรเจคคือ :
        ทำให้อัลกอริทึมที่เป็นทฤษฎี กลายเป็นสิ่งที่มองเห็น ทดลอง และเข้าใจได้จริง
        ฝึกคิดเชิงตรรกะ
        ฝึกออกแบบ Algorithm
        เตรียมพื้นฐาน Software Development
        สร้างระบบที่อธิบาย Algorithm ได้แบบ Visual

    ฟีเจอร์หลักในโปรเจค :
        โปรเจคประกอบด้วยหลายโมดูล เช่น :
            Tower of Hanoi Interactive Lab :
                แก้โจทย์ด้วยตัวเอง (Drag & Drop)
                AI Recursive Solver
                แสดง Time Complexity และ Stack Behavior

            Pathfinding Visualizer (A / Dijkstra)* :
                จำลองการค้นหาเส้นทางบน Grid
                แสดงการทำงานของ Priority Queue และ Min-Heap

            Data Structure Lab :
                Linked List :
                Stack :
                Heap :

            Algorithm Mini Tools :
                Caesar Cipher
                Anagram Checker
                Calculator (Infix → Postfix Logic)
                Pattern Generator

    Backend System :
        พัฒนา API ด้วย :
            FastAPI (Python)
            CORS Middleware
            File Download Endpoint
            Dynamic File Serving System
        ตัวอย่าง Endpoint :
            GET /download?file=filename

    สิ่งที่โปรเจคนี้สะท้อนทักษะ :
        เข้าใจ Algorithm และ Data Structure ระดับพื้นฐานถึงกลาง
        เขียน Recursive Algorithm จริง
        เข้าใจ Big-O (Time / Space Complexity)
        ออกแบบโครงสร้าง OOP ใน JavaScript
        แยก Algorithm Logic ออกจาก UI
        ออกแบบระบบแบบ Modular
        พัฒนา REST API ด้วย FastAPI
        เข้าใจโครงสร้างโปรเจคแบบ Frontend + Backend

Caesar Cipher & Anagram Tool :

    ส่วนหนึ่งของโปรเจค: The Algorithm Vault

    ภาพรวมโปรเจค :
        เครื่องมือทดลองด้าน Algorithm และการจัดการข้อความ (String Processing)
        พัฒนาเพื่อฝึกการเขียนตรรกะ การออกแบบโครงสร้างโปรแกรม และวิเคราะห์ประสิทธิภาพการทำงาน

    ระบบประกอบด้วย :
        เข้ารหัสข้อความ (Caesar Cipher)
        ถอดรหัสข้อความ
        ตรวจสอบคำว่าเป็น Anagram หรือไม่
        บันทึกประวัติการใช้งาน (History Log)
        แสดงกระบวนการคำนวณแบบ Real-time (Logic Monitor)

    จุดประสงค์ของโปรเจค :
        ฝึกการออกแบบอัลกอริทึม
        ฝึกจัดการข้อมูลตัวอักษร
        ฝึกใช้ Modular Arithmetic
        ฝึกวิเคราะห์ Time Complexity (Big-O)
        ฝึกแยกส่วน UI และ Logic อย่างชัดเจน

    เทคโนโลยีที่ใช้  :
        HTML5
        CSS3 (Grid Layout)
        JavaScript (Vanilla JS)
        localStorage สำหรับจัดเก็บข้อมูลชั่วคราว

    ฟังก์ชันหลัก :
        Caesar Cipher
        รองรับตัวอักษรพิมพ์เล็กและพิมพ์ใหญ่
        รองรับค่า Shift ติดลบ
        ใช้ Modular Arithmetic ป้องกันค่าล้น
        ความซับซ้อนการทำงาน: O(n)

    Anagram Checker :
        เปรียบเทียบคำ 2 คำว่ามีตัวอักษรเหมือนกันหรือไม่
        ใช้วิธีจัดเรียงตัวอักษร (Sorting)
        ความซับซ้อนการทำงาน: O(n log n)

    History Log :

        เก็บประวัติการใช้งานล่าสุด 10 รายการ
        จัดเก็บผ่าน localStorage
        แสดงผลแบบ Real-time

    โครงสร้างไฟล์ :
        CaesarCipher_And_AnagramChecker.html
        scriptsCip.js
        CipStyle.css

        HTML → โครงสร้างหน้าเว็บ
        JavaScript → ควบคุมตรรกะทั้งหมด
        CSS → ออกแบบ UI แบบ Dark Theme

    สิ่งที่ได้เรียนรู้จากโปรเจคนี้ :
        การออกแบบฟังก์ชันให้รองรับ Edge Case
        การจัดการ State ของโปรแกรม
        การแยกหน้าที่การทำงาน (Separation of Concerns)
        การวิเคราะห์ประสิทธิภาพการทำงานของอัลกอริทึม

    เหมาะสำหรับ :
        ใช้เป็นสื่อการสอนพื้นฐาน Algorithm
        ใช้เป็นตัวอย่างฝึกเขียนตรรกะ
        ใช้ต่อยอดพัฒนา Cryptography ขั้นสูง

    ผู้พัฒนา :

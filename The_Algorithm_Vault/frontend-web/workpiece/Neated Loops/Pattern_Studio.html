Visual Pattern Analysis: Nested Loops

การวิเคราะห์พิกัด 2 มิติและการประยุกต์ใช้ตรรกะคณิตศาสตร์ในการสร้างรูปทรง

1. นิยามของความสัมพันธ์ในพิกัด 2 มิติ

การสร้างรูปทรงบน Grid ขนาด n × n คือการทำงานบนระบบพิกัดแบบ Cartesian Discrete ซึ่งประกอบด้วยสองดัชนีหลัก:

  • Outer Loop: ควบคุมแถว (i)
  • Inner Loop: ควบคุมคอลัมน์ (j)

ทุกจุดในกริดสามารถนิยามเป็นฟังก์ชันบูลีน:

G(i,j) ∈ {0,1}

2. จุดกึ่งกลาง (Midpoint Logic)

mid = floor(n / 2)

ใช้เป็นแกนอ้างอิงของรูปทรงสมมาตร เช่น Diamond, Pyramid

3. Taxonomy ของ Pattern Logic

ประเภท สมการ ตัวอย่าง
Full Fill true Square
Boundary i=0 ∨ i=n-1 ∨ j=0 ∨ j=n-1 Hollow Square
Linear Inequality j ≤ i Triangle
Distance Based |i-mid|+|j-mid| ≤ mid Diamond

4. Manhattan Distance vs Euclidean

ระบบนี้ใช้ Manhattan Distance:

|x₁-x₂| + |y₁-y₂|

เพราะคำนวณเร็วกว่าและเหมาะกับ Grid แบบ Discrete

5. Formal Computational Model

G(i,j) = F(i,j,n)

Nested Loop คือการ evaluate F สำหรับทุก cell 


for (let i = 0; i < n; i++) {
    for (let j = 0; j < n; j++) {
        G[i][j] = F(i,j,n);
    }
}

6. Correctness Proof Sketch

Outer loop ทำ n ครั้ง Inner loop ทำ n ครั้ง

Total = n × n = n²

ดังนั้น algorithm ครอบคลุมทุก cell โดยไม่ซ้ำ

7. Time Complexity Analysis

T(n) = n²
  • Worst-case = O(n²)
  • Average-case = O(n²)

8. Memory Complexity

ใช้โครงสร้าง:


gridState = Array(n*n)
steps = Array(n*n)
M(n) = O(n²)

หากไม่เก็บ steps → ลดเหลือ O(n²) หาก render streaming → O(1)

9. Animation as State Machine

S_k = (i_k, j_k, grid_k)

Transition ต่อเนื่องจน k = n² − 1

The Algorithm Vault: จากตรรกะพื้นฐานสู่สถาปัตยกรรมซอฟต์แวร์

พื้นที่แห่งนี้คือบันทึกการเดินทางผ่านตัวอักษรและตรรกะ รวบรวมองค์ความรู้ด้านอัลกอริทึมและโครงสร้างข้อมูลที่กลั่นกรองมาจากประสบการณ์การเรียนรู้และการพัฒนา จากโจทย์พื้นฐานในห้องเรียน สู่การออกแบบระบบที่มีโครงสร้างแบบ Clean Architecture เพื่อพิสูจน์ว่าซอฟต์แวร์ที่ดีไม่ได้มีแค่ฟังก์ชันที่ทำงานได้ แต่ต้องมีรากฐานที่ยืดหยุ่นและยั่งยืน หอนี้รวบรวมโปรเจกต์อัลกอริทึมและโครงสร้างข้อมูลที่ผมพัฒนาขึ้น โดยมุ่งเน้นที่การประยุกต์ใช้ความรู้พื้นฐานสู่โครงสร้างระบบที่ยืดหยุ่นภายใต้แนวคิดClean Architecture

The Algorithm Vault

บันทึกการเดินทางจากตรรกะพื้นฐานสู่สถาปัตยกรรมซอฟต์แวร์

Pathfinding Visualizer

การแสดงผล Dijkstra & A* อัลกอริทึมในการค้นหาเส้นทางที่สั้นที่สุด

Data Structure Lab

จำลองการทำงานของ Linked List และ Stack ที่สร้างขึ้นเองด้วย Java

Math Expression Engine

ระบบแปลง Infix เป็น Postfix และการคำนวณนิพจน์คณิตศาสตร์ที่ซับซ้อน

Visual Pattern Studio

การสร้างงานศิลปะเชิงเรขาคณิตด้วย Nested Loops และตรรกะคณิตศาสตร์

Security & Utility Tools

เครื่องมือเข้ารหัสลับ Caesar Cipher และระบบตรวจสอบ Anagram

Towers of Hanoi

การแก้ปัญหาระดับตำนานด้วย Recursion พร้อมแอนิเมชันจำลองการย้ายดิสก์
